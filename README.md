# snak

> Sats-staked snake battle royale. Last serpent alive takes the pool.

Onchain snake matches with cUSD entry fees and prize-pool settlement on Celo.

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript

## Develop

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
